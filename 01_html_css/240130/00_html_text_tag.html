텍스트와 관련한 태그들

강조

strong 태그

설명

strong 태그는 해당 내용의 중요, 심각 또는 긴급성을 강하게 강조(strong important)하는데 사용되는 태그이다.
제목, 캡션, 문단 등에서 중요한 부분을 굵은 글꼴로 표시하여 일반적인 텍스트와 구별한다.

주의사항

단순히 굵은 글꼴 두께를 사용하기 위해서 strong 태그를 사용하지 말자.
strong 태그는 중요성을 나타낼 때 사용되어야 한다.
스타일을 위해서는 CSS(font-weight:bold)를 사용해야 한다.

em 태그

설명

em 태그는 주위 텍스트에 비해 문맥적으로 중요한 내용을 강조(stress emphasis)하는 태그이다.
대부분의 브라우저에서는 em 태그는 텍스트를 기울임체로 표현한다.

주의사항

단순히 기울임체를 사용하기 위해서 em 태그를 사용하지 말자.
em 태그는 중요성을 나타낼 때 사용되어야 한다.
스타일을 위해서는 CSS(font-style:italic)을 사용해야 한다.

mark 태그

설명

mark태그는 주위 텍스트에 비해 중요한 내용을 표시(mark)하는 태그이다.
대부분의 브라우저에서는 mark 태그의 텍스트를 형광펜으로 표시한다.

주의사항

단순히 형광펜 스타일을 사용하기 위해서 mark 태그를 사용하지 말자.
mark 태그는 중요성을 나타낼 때 사용되어야 한다.
형광펜 스타일을 위해서는 CSS(background-color:yellow)를 사용해야 한다.

변경

ins 태그

설명

ins 태그는 문서에 추가(insert)된 텍스트를 나타내는 태그이다.
대부분의 브라우저에서는 ins 태그의 텍스트를 밑줄로 표시한다.
주로 third party에서 제공하는 광고 태그로 자주 쓰인다.

del 태그

설명

del 태그는 문서에서 삭제(delete)된 텍스트를 나타내는 태그이다.
대부분의 브라우저에서는 del 태그의 텍스트를 밑줄이 아니라 기울임체도 아니라 취소선으로 표시한다.

cite와 datetime 속성

설명

cite 속성은 추가, 삭제된 내용에 대한 설명을 제공하는 리소스이다.
datetime 속성은 추가, 삭제된 내용의 시간 및 날짜를 나타낸다.

첨자

sup, sub 태그

설명

sup 태그는 주위 텍스트에 비해 상위 첨자(superscript)로 표현되는 텍스트를 나타내는 태그이다.
sub 태그는 주위 텍스트에 비해 상위 첨자(subscript)로 표현되는 텍스트를 나타내는 태그이다.

주의사항

sup, sub 태그는 단순한 표시, 모양 목적 등의 시각적 표현을 위한 것이 아니다.
타이포그래피적 규칙이나 관례 상 첨자로 표시되는 지수, 서수, 각주 번호, 변수 첨자, 화학식 등의 의미를 맞추기 위해 사용된다.

인용

q 태그

설명

q 태그는 짧은 인용문(quotation)을 나타내는 태그이다.
짧은 인용문의 시작과 끝에 따옴표가 자동으로 삽입된다.

blockquote 태그

설명

blockquote 태그는 긴 인용문을 나타내는 태그이다.
긴 인용문은 블록 요소로 CSS의 margin 값이 기본으로 적용된다.

주의사항

단순히 들여쓰기 스타일을 위해 사용하지 맙시다.
들여쓰기를 위해서는 CSS(margin-left)를 사용해야 합니다.
cite 속성을 사용하면 인용문의 출처를 제공할 수 있고, 여러 곳에서 이 정보를 활용할 수 있습니다.

기타

abbr 태그

설명

abbr 태그는 약어(abbreviation)를 나타낸다.
abbr 태그는 title 속성을 이용하기 적합하다.

address 태그

설명

address 문서의 작성자, 소유자의 연락처 정보를 나타내는 태그이다.
대부분의 브라우저에서는 기울임체로 표현된다.

dfn 태그

설명

dfn 태그는 용어의 정의(definition)를 나타내는 태그이다.

cite 태그

설명

cite 태그는 창작물의 제목을 나타내는 태그이다.
BTS-DNA

code 태그

설명

code 태그는 컴퓨터 코드를 나타내는 태그이다.
대부분의 브라우저에서는 code 태그의 텍스트를 고정폭 글꼴로 표현한다.

주석(comment)

설명

주석은 브라우저에 표시되지 않는 텍스트이다.
주로 코드에 대한 설명이나 메모를 작성할 때 사용한다.

엔티티(Entity)

설명

HTML에 미리 예약된 예약어를 통해, 웹브라우저가 평소와 다른 의미로 그것을 해석하게 할 수 있다.
&를 이용하여 <, >, &,  ,  ,  등으로 사용할 수 있다.
